ĆWICZENIE 8: WDRÓŻ APLIKAC J Ę ZA POMOCĄ USŁUGI ELASTIC BEANS TALK
Amazon Elastic Beanstalk to łatwa w użyciu usługa do wdrażania i skalowania aplikacji i
usług internetowych opracowanych w językach Java, .NET, PHP, Node.js, Python, Ruby, Go i
Docker na znanych serwerach, takich jak Apache, Nginx, Passenger i IIS.
1. Przejdź do usługi Elastic Beanstalk
2. Wybierz Create Application.
3. Wpisz nazwę aplikacji.
4. Na karcie Platform wybierz Node.js i najnowszą dostępną wersję.
5. Na karcie Application code wybierz Upload your code.
6. Dla twoich szkoleniowych potrzeb wybierz ze strony https://docs.aws.amazon.com/
elasticbeanstalk/latest/dg/tutorials.html przykład z Node.js (https://
docs.aws.amazon.com/elasticbeanstalk/latest/dg/samples/nodejs.zip).
7. Rozpakuj plik do katalogu beanstalk. Możesz zmienić treść pliku index.html
8. Spakuj plik zipem.
9. Wgraj plik zip i wybierz Create Application.
